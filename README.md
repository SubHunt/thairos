# Thairos – Оптово-розничная продажа пива и напитков

Проект веб-сайта для компании Thairos, эксклюзивного импортёра и дистрибьютора премиального пива из Таиланда, Вьетнама, Японии, Сингапура и других стран на территории России.

## Технологии

- **Backend**: Django 5
- **База данных**: PostgreSQL
- **Frontend**: TailwindCSS (через CDN), Swiper.js для каруселей
- **Язык**: Python 3.12
- **Сервер**: Разработка – встроенный сервер Django, продакшен – gunicorn + nginx (планируется)

## Структура проекта

```
thairos/
```

## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd thairos
```

### 2. Создание виртуального окружения
```bash
python -m venv venv --without-pip
# Активация (Windows)
venv\Scripts\activate
# Установка pip
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python get-pip.py
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка базы данных
Создайте базу PostgreSQL и настройте подключение в `thairos_site/settings.py` (секция DATABASES).

Выполните миграции:
```bash
python manage.py migrate
```

### 5. Запуск сервера разработки
```bash
python manage.py runserver
```
Сайт будет доступен по адресу: http://127.0.0.1:8000

## Основные функции

### Age Gate (возрастное подтверждение)
При первом посещении пользователь видит страницу с подтверждением возраста. После принятия устанавливается cookie, и доступ к сайту открывается.

### Главная страница
- Hero-секция с призывом к действию
- Категории продукции (пиво, напитки, снеки) с изображениями
- Карусель брендов с автоматической прокруткой
- Форма «Запросить прайс» с модальным окном успешной отправки
- Футер с контактами и ссылками

### Страницы брендов
Каждый бренд (Singha, Leo, Saigon, Tiger, Sapporo и др.) имеет отдельную страницу с детальным описанием и фирменным фоном.

### Административная панель
Доступна по адресу `/admin` для управления контентом (категории, бренды, продукты).

## Карусель брендов

Используется Swiper.js с настройками:
- Автопрокрутка каждые 3 секунды
- Остановка при наведении
- Адаптивное количество слайдов (2 на мобильных, 5 на десктопе)
- Круглые фоны с логотипами, вписанными в границы

## Форма обратной связи

Форма «Запросить прайс» включает поля:
- Имя
- Компания
- Телефон
- E-mail
- Сообщение

После отправки показывается модальное окно с подтверждением.
